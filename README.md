# PhantomTerm

**Free Windows tool** that boots supported Linux live ISOs directly into **Windows Terminal** — no VM window, no installation, no persistence.

### Features
- Pure text Linux terminal (no GUI)
- Automatic login (no password required)
- **New random MAC address + fresh IP every launch**
- Completely ephemeral sessions
- Very lightweight

### Supported Distros (Auto-login works)
- Ubuntu
- Linux Mint (Cinnamon, MATE, Xfce)
- Zorin OS
- Kubuntu, Lubuntu, Xubuntu, Ubuntu MATE

### Not Supported for Auto-login
Kali Linux, Parrot OS, Tails (tool will show warning)

### Requirements
- Windows 10 or 11
- `qemu-system-x86_64.exe` from https://qemu.weilnetz.de/w64/ (must be in same folder as PhantomTerm.exe)
- Windows Terminal (recommended)
